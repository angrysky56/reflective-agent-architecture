## 4. The Unified Theory

Agents follow minimal, context-sensitive heuristics without central control:

| Domain            | Local Rule                                    |
| ----------------- | --------------------------------------------- |
| Ants              | "Follow strongest pheromone trail"            |
| Neurons           | "Fire if input exceeds threshold"             |
| Traders           | "Buy low, sell high based on perceived value" |

#### Pattern 2: Feedback Loops (Reinforcing + Balancing)

Outputs of interactions feed back as inputs:

- **Positive feedback**: Pheromone reinforcement, price momentum, neural potentiation
- **Negative feedback**: Pheromone evaporation, market corrections, inhibitory neurons

#### Pattern 3: Nonlinearity

Small changes produce disproportionately large effects:

- Sigmoidal activation functions
- Phase transitions at critical thresholds
- Power-law distributions in outcomes

#### Pattern 4: Recursive Self-Reference

Systems form closed loops that refer to their own states:

- Ant colonies model their environment through distributed pheromone maps
- Markets price in expectations about future prices
- Consciousness models itself (meta-cognition)
